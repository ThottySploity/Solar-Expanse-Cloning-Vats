# Cloning Vats

A mod for Solar Expanse that introduces artificial population growth through cloning technology.

## Overview

Cloning Vats adds a new way to increase colony population by growing colonists in specialized facilities. Instead of relying entirely on buying populations, your colonies can invest resources into cloning infrastructure to accelerate expansion across the Solar System.

Perfect for developing remote outposts, supporting industrial megacolonies, or jump-starting new settlements on hostile worlds.

## Features

- New Cloning Vat building
- Artificial population growth
- Resource-driven colonist production
- Faster colony development
- Income source

## Installation

This mod is depended on the Teddit mod for Solar Expanse

- Follow the instructions of the Teddit mod, and place these files into the "mod" folder under a new folder name called: "Cloning Vats" 
- Launch the game

## Notes

- Works great with the "Waste" mod!
